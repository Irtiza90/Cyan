# Cyan Programming Language

**Status**: Unstable

## Current features

- variables
- binary and unary operations
- if/else expressions
- while loops
- functions
- lambda functions

## Usage

Pre-requiestes: `git` and `python3`

Commands:
```bash
git clone https://github.com/was07/Cyan
python Cyan
```
It will start the Cyan shell.

## Example Code

```js
Cyan 1.0.0 shell on win32
>>> 7 ** 2
49
>>> fun square(i): i ** 2
<Function square>
>>> square(4)
16
>>> let a = 2
2
>>> if a != 2 then 100 else 200
200
>>> print(a)
2   
None
>>> while a < 5 {let a = a + 1}
None
```
