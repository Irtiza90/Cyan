# Cyan Programming Language

**Status**: Unstable

**Current features**
- variables
- binary and unary operations
- if/else expressions
- while loops
- functions
- lambda functions

__

## Usage

Pre-requiestes: `git` and `python3`

Commands:
```bash
git clone https://github.com/was07/Cyan
python Cyan
```
It will start the Cyan shell.

__

## Example Code

```
Cyan 1.0.0
&> fun sq(i): i ^ 2
<Function sq>
&> sq(8)
64
&> let a = 1
1
&> if a > 2 then 4 else 9
9
```
